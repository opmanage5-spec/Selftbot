Voice Auto

Rejoindre un salon vocal automatiquement
Fais clic droit sur un salon vocal → Copier l'identifiant (Mode développeur requis)
Comment activer le mode développeur ?
Paramètres Discord → Apparence → Activer le mode développeur.
Ensuite, clic droit sur n'importe quel salon → Copier l'identifiant.

DM All

Envoyer un message privé à tous tes amis
{user} sera remplacé par le pseudo de chaque destinataire. Mode anti-détection actif : délai aléatoire (2–4.5s), indicateur de frappe, et pause automatique en cas de captcha.
2s
Délai entre chaque DM. Plus c'est élevé, moins tu risques un rate-limit.
Attention
L'envoi massif de DM peut entraîner des restrictions sur ton compte Discord. Utilise cette fonction avec modération.

DM Serveur

Envoyer un DM à tous les membres d'un serveur
Le selfbot doit être membre de ce serveur. Clic droit sur le serveur → Copier l'identifiant.
{user} sera remplacé par le pseudo de chaque membre. Mode anti-détection actif : délai aléatoire, indicateur de frappe, résolution automatique des captchas.
2s
Délai entre chaque DM. Plus c'est élevé, moins tu risques un rate-limit.
Attention
Les membres avec les DMs désactivés seront ignorés automatiquement. L'envoi massif peut prendre du temps selon la taille du serveur.
